Задача 1: Запасы патронов Гордона

История:

Гордон Фримен обнаружил склад с ящиками боеприпасов. Каждый ящик содержит 12 патронов. Он нашёл 5 таких ящиков.

Вопрос:

Сколько всего патронов собрал Гордон?

Решение:

  1. Понимание задачи:

    Нам нужно найти общее количество патронов, которые собрал Гордон, зная количество ящиков и количество патронов в каждом ящике.

  2. Известные данные:

    • Количество ящиков: 5
    • Патронов в одном ящике: 12

  3. План решения:

    Чтобы найти общее количество патронов, нужно умножить количество ящиков на количество патронов в одном ящике.

  4. Выполнение вычислений:

    • Шаг 1: Умножаем количество ящиков на количество патронов в одном ящике. Общее количество патронов=Количество ящиков×Патронов в одном ящике\text{Общее количество патронов} = \text{Количество ящиков} \times \text{Патронов в одном ящике}Общее количество патронов=Количество ящиков×Патронов в одном ящике Общее количество патронов=5×12\text{Общее количество патронов} = 5 \times 12Общее количество патронов=5×12
    • Шаг 2: Вычисляем произведение. 5×12=605 \times 12 = 605×12=60

  5. Ответ:

    Гордон собрал всего 60 патронов.

Задача 2: Делёж крышек

Сколько крышек получит каждый член команды?

  2. Известные данные:

    • Общее количество крышек: 84

  3. План решения:

    Разделим общее количество крышек на количество членов команды, чтобы найти количество крышек на одного человека.

  4. Выполнение вычислений:

    • Шаг 1: Разделяем общее количество крышек на количество членов команды. Крышек на одного человека=Общее количество крышекКоличество членов команды\text{Крышек на одного человека} = \frac{\text{Общее количество крышек}}{\text{Количество членов команды}}Крышек на одного человека=Количество членов командыОбщее количество крышек​ Крышек на одного человека=847\text{Крышек на одного человека} = \frac{84}{7}Крышек на одного человека=784​
    • Шаг 2: Вычисляем результат деления. 847=12\frac{84}{7} = 12784​=12

  5. Ответ:

    Каждый член команды получит по 12 крышек.

Задача 3: Производство стимпаков

История:

В убежище учёные могут изготовить 3 стимпака за 2 часа.

Вопрос:

Сколько стимпаков они смогут произвести за 8 часов?

Решение:

  1. Понимание задачи:

    Нужно определить общее количество стимпаков, которое учёные могут произвести за 8 часов, зная производительность за 2 часа.

  2. Известные данные:

    • Количество стимпаков за 2 часа: 3
    • Общее время производства: 8 часов

  3. План решения:

    • Определим, сколько 2-часовых периодов содержится в 8 часах.
    • Затем умножим количество периодов на количество стимпаков, производимых за каждый период.

  4. Выполнение вычислений:

    • Шаг 1: Находим количество 2-часовых периодов в 8 часах. Количество периодов=Общее времяДлительность одного периода\text{Количество периодов} = \frac{\text{Общее время}}{\text{Длительность одного периода}}Количество периодов=Длительность одного периодаОбщее время​ Количество периодов=82=4\text{Количество периодов} = \frac{8}{2} = 4Количество периодов=28​=4
    • Шаг 2: Вычисляем общее количество стимпаков. Общее количество стимпаков=Количество периодов×Стимпаков за период\text{Общее количество стимпаков} = \text{Количество периодов} \times \text{Стимпаков за период}Общее количество стимпаков=Количество периодов×Стимпаков за период Общее количество стимпаков=4×3=12\text{Общее количество стимпаков} = 4 \times 3 = 12Общее количество стимпаков=4×3=12

  5. Ответ:

    Учёные смогут произвести 12 стимпаков за 8 часов.

Задача 4: Энергия силовых доспехов

История:

Братство Стали готовится к бою и заряжает свои силовые доспехи. Каждый доспех требует 4 энергетических батареи. Если у них есть 9 доспехов,

Вопрос:

Сколько батарей им необходимо?

Решение:

  1. Понимание задачи:

    Нужно вычислить общее количество батарей, необходимых для зарядки всех доспехов.

  2. Известные данные:

    • Количество доспехов: 9
    • Батарей на один доспех: 4

  3. План решения:

    Умножим количество доспехов на количество батарей, требуемых для одного доспеха.

  4. Выполнение вычислений:

    • Шаг 1: Вычисляем общее количество батарей. Общее количество батарей=Количество доспехов×Батарей на один доспех\text{Общее количество батарей} = \text{Количество доспехов} \times \text{Батарей на один доспех}Общее количество батарей=Количество доспехов×Батарей на один доспех Общее количество батарей=9×4\text{Общее количество батарей} = 9 \times 4Общее количество батарей=9×4
    • Шаг 2: Вычисляем произведение. 9×4=369 \times 4 = 369×4=36

  5. Ответ:

    Братству Стали необходимо 36 батарей для зарядки всех доспехов.

Задача 5: Поездка на бронепоезде

История:

Герой должен проехать на бронепоезде 360 километров до следующей базы. Если поезд движется со скоростью 60 километров в час,

Вопрос:

Сколько времени займёт поездка?

Решение:

  1. Понимание задачи:

    Необходимо найти время, за которое поезд преодолеет заданное расстояние при данной скорости.

  2. Известные данные:

    • Расстояние: 360 км
    • Скорость поезда: 60 км/ч

  3. План решения:

    Используем формулу для вычисления времени в пути:

    Время=РасстояниеСкорость\text{Время} = \frac{\text{Расстояние}}{\text{Скорость}}Время=СкоростьРасстояние​

  4. Выполнение вычислений:

    • Шаг 1: Подставляем известные значения в формулу. Время=360 км60 км/ч\text{Время} = \frac{360 \text{ км}}{60 \text{ км/ч}}Время=60 км/ч360 км​
    • Шаг 2: Вычисляем результат деления. Время=6 часов\text{Время} = 6 \text{ часов}Время=6 часов

  5. Ответ:

    Поездка займёт 6 часов.

Задача 6: Сбор Нука-Колы

История:

Избранный собирает бутылки Нука-Колы для торговли. В первый день он собрал 18 бутылок, во второй день — на 6 бутылок больше, чем в первый.

Вопрос:

Сколько бутылок Нука-Колы Избранный собрал за два дня?

Решение:

  1. Понимание задачи:

    Необходимо определить общее количество бутылок, собранных за два дня, учитывая, что во второй день было собрано на 6 бутылок больше, чем в первый.

  2. Известные данные:

    • Количество бутылок в первый день: 18
    • Во второй день собрано на 6 бутылок больше, чем в первый

  3. План решения:

    • Вычислим количество бутылок, собранных во второй день.
    • Суммируем количество бутылок за оба дня.

  4. Выполнение вычислений:

    • Шаг 1: Определяем количество бутылок во второй день. Бутылок во второй день=Бутылок в первый день+6\text{Бутылок во второй день} = \text{Бутылок в первый день} + 6Бутылок во второй день=Бутылок в первый день+6 Бутылок во второй день=18+6=24\text{Бутылок во второй день} = 18 + 6 = 24Бутылок во второй день=18+6=24
    • Шаг 2: Суммируем количество бутылок за два дня. Общее количество бутылок=Бутылок в первый день+Бутылок во второй день\text{Общее количество бутылок} = \text{Бутылок в первый день} + \text{Бутылок во второй день}Общее количество бутылок=Бутылок в первый день+Бутылок во второй день Общее количество бутылок=18+24=42\text{Общее количество бутылок} = 18 + 24 = 42Общее количество бутылок=18+24=42

  5. Ответ:

    Избранный собрал всего 42 бутылки Нука-Колы за два дня.
